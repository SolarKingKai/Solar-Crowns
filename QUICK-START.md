# Solar Crowns - Quick Start Guide

## Fastest Way to Launch

```bash
./launch.sh
```

This opens an interactive menu showing all available browsers on your system.

---

## All Launch Options

### 1. Universal Launcher (Recommended)

**Interactive Menu:**
```bash
./launch.sh
```

**Direct Launch:**
```bash
./launch.sh brave      # Open in Brave
./launch.sh edge       # Open in Microsoft Edge
./launch.sh firefox    # Open in Firefox
./launch.sh chrome     # Open in Chrome
```

---

### 2. Individual Browser Scripts

Click to run or execute in terminal:

| Browser | Command |
|---------|---------|
| Brave Browser | `./open-in-brave.sh` |
| Microsoft Edge | `./open-in-edge.sh` |
| Firefox | `./open-in-firefox.sh` |
| Google Chrome | `./open-in-chrome.sh` |

---

### 3. Direct Browser Commands

```bash
brave index.html
microsoft-edge index.html
firefox index.html
google-chrome index.html
```

---

### 4. Double-Click Method

Simply double-click `index.html` and choose your browser when prompted.

## Troubleshooting

**Browser not found?**
- Install your preferred browser first
- Run `./launch.sh` to see which browsers are detected

**Script won't run?**
- Make sure scripts are executable: `chmod +x *.sh`
- Run from the website directory
